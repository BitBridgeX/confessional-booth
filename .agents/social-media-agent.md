**Reporting Dashboard** (Supabase):
```sql
CREATE TABLE cb_social_metrics (
  id UUID PRIMARY KEY,
  platform TEXT,
  date DATE,
  followers INT,
  views INT,
  engagements INT,
  clicks INT,
  revenue DECIMAL,
  created_at TIMESTAMP
);

-- Query: Monthly growth per platform
SELECT platform, 
       COUNT(*) as posts, 
       SUM(views) as total_views,
       SUM(engagements) as total_engagements,
       SUM(revenue) as revenue
FROM cb_social_metrics
WHERE date >= DATE_SUB(NOW(), INTERVAL 1 MONTH)
GROUP BY platform;
```
