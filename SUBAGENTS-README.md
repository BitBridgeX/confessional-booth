### 5. 🎮 Domagotchi Game Agent
**File**: `.agents/domagotchi-game-agent.md` (473 lines)

**Responsibilities**:
- NLP confession processing engine
- Sin/virtue scoring system
- Gamification mechanics (levels, badges, progression)
- Haptic feedback integration
- Couples mode + leaderboards

**Core Mechanics**:

**Confession Processing** (NLP):
- User confesses → System extracts intent (lust, pride, gluttony, etc.)
- Authenticity scoring (how detailed + genuine)
- Sin points calculated (1-100 per confession)
- Penance options auto-generated (4 types)

**Gamification**:
- **Levels**: 1-100 with progressive XP thresholds
- **Ranks**: Penitent → Confessor → Saint/Sinner → Domme/Sub
- **Badges**: 50+ (confession milestones, penance mastery, sin celebrations, seasonal)
- **Progression**: Level-up animations, reward unlocks, content gating

**Sin/Virtue System**:
- **Sin Score**: 0-1000 (accumulates per confession)
- **Virtue Score**: 0-1000 (completion of penance)
- **Balance**: Overall alignment (sinner vs. saint scale)
- **Decay**: Sin naturally decreases over time; penance accelerates decay

**Penance Mechanics** (4 Types):
1. **Challenge**: Multi-day behavioral challenges ("7-day celibacy," "journaling challenge")
2. **Service**: Act of service to others (donation, help someone, apologize)
3. **Ritual**: Daily spiritual practices (meditation, prayer, blessing ritual)
4. **Behavioral**: Redirect bad behaviors into productive ones

**Haptic Integration**:
- Confession submitted → buzz (acknowledgment)
- Penance assigned → pulse series (anticipation)
- Achievement unlocked → vibration crescendo (celebration)
- Boss defeated → long vibration (triumph)
- High sin warning → stuttering vibration (alarm)
- Couples sync → synchronized vibration (intimacy)

**Couples Mode**:
- Partnership system (level 1-50)
- Role assignment (domme/sub)
- Synchronized challenges + power dynamics
- Shared leaderboards + tournaments
- Haptic sync between devices

**Boss Challenges** (Weekly/Monthly):
- **The Puritan** (Med): 7-day abstinence challenge
- **The Succubus** (Hard): 3-day lean-into-desires challenge
- **The Mirror** (Expert): 1000-word fear confession
- **The Domme** (Expert, Couples): Partner-assigned penance
- And more...

**Phase 1 Deliverables**:
- Game design document (complete mechanics, balancing, curves)
- NLP pipeline (confession processing, intent extraction, sin calculation)
- Database schema (users, confessions, penances, badges, leaderboards, couples)
- 5 boss challenges (designed + balanced + rewards)
- Haptic integration spec (events, patterns, device support)
- Couples mode design (partnership, challenges, synchronization)

**Success Metrics**:
- User retention: 60% day 7
- Daily active users: 80%+ of registered
- Average confessions: 3+ per week per user
- Penance completion: 70%+
- Couples adoption: 20%+ of active users
