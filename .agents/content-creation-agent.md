## Production Workflow

### Sermon/Podcast Production
1. **Concept**: Write outline + thesis
2. **Script**: Full script (aim for natural delivery)
3. **Record**: Audio recording (high-quality mic + quiet space)
4. **Edit**: Audio mixing + transitions + music
5. **Visual**: (If video) Animate COOKIE MAMI or create visual elements
6. **Package**: Create transcript + show notes + study guide
7. **Publish**: Upload to Supabase + distribute to platforms

### Hymn Production
1. **Composition**: Write lyrics + melody structure
2. **Generate**: Use Suno AI to generate base version
3. **Refine**: Layer with live instrumentation (if budget allows)
4. **Lyrics**: Finalize sacred words
5. **Publish**: Create lyric video + audio versions

### Illustrated Book Production
1. **Write**: Draft all text content
2. **Illustrate**: Commission or create illustrations
3. **Design**: Layout + typesetting (beautiful print-ready PDF)
4. **Package**: Create print + digital versions
5. **Distribute**: Upload to merch store as premium digital product + print-on-demand
