## Content Categories

### Songs (`songs/`)

**Hymns** (`hymns/`)
- Original sacred compositions
- Suno AI generated + refined
- Liturgical structure with modern production
- Themes: sensuality, dominance, liberation, embodied faith

**Worship** (`worship/`)
- Devotional music content
- Background tracks for meditation/ritual
- Atmospheric + contemplative

**Secular** (`secular/`)
- Cultural references
- Ambient + experimental
- Crossover tracks

**Metadata**: Each song includes:
- Title, composer, date
- Lyrics (markdown)
- Audio file (URL or stored)
- Suno ID (if AI-generated)
- Duration, BPM, key
- Usage rights

### Sermons (`sermons/`)

**"AMEN, SLUT" Series** (`series-amen-slut/`)
- 12-episode foundational theology series
- Topics: sacred body, power dynamics, confession, pleasure, shame deconstruction
- Format: 15-45 min per episode
- Includes: video, audio, transcript, study guide

**Transcripts** (`transcripts/`)
- Full sermon text (markdown)
- Timestamped sections
- Key quotes extracted
- Bible/text references

**Audio** (`audio/`)
- Sermon recordings (WAV, MP3)
- Video metadata (Mux IDs)
- Production notes

**Metadata**: Each sermon includes:
- Title, speaker, date
- Series + episode number
- Duration, recording format
- Transcript location
- Video playback ID
- Study guide link
- Themes/tags

### Ceremonies (`ceremonies/`)

**Confessional Rituals** (`confessional-rituals/`)
- Step-by-step confession practices
- Preparation guidelines
- Penance options (challenge, service, ritual, behavioral)
- Closure/blessing scripts

**Blessing Ceremonies** (`blessing-ceremonies/`)
- Sacred moments for personal milestones
- Group blessing rituals
- Seasonal ceremonies
- Life transition rituals

**Community Rituals** (`community-rituals/`)
- Shared accountability practices
- Group confessions
- Collective blessing moments
- Live stream ceremony formats

**Metadata**: Each ceremony includes:
- Purpose + intention
- Duration + preparation time
- Required items/props
- Step-by-step script
- Music/audio recommendations
- Accessibility notes

### Scripture (`scripture/`)

**Gospel** (`gospel/`)
- HolyFans reimagined biblical stories
- Modern spiritual interpretations
- Themes: sexuality as sacred, consent, power, liberation
- Format: 500-1000 words per story

**Prayers** (`prayers/`)
- Daily/weekly devotionals
- Morning affirmations
- Evening reflections
- Crisis prayers
- Format: 200-400 words, poetic language

**Mantras** (`mantras/`)
- "AMEN, SLUT" philosophy statements
- Affirmations for embodied faith
- Shame-breaking declarations
- Power-claiming statements

**Metadata**: Each scripture piece includes:
- Title + author
- Date created
- Theme/category
- Word count
- Related sermon/content
- Audio recording (if available)

6. **Keep files organized** by category
