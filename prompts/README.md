## How to Use

### For Designers/Creators
1. Navigate to relevant folder (midjourney, video, or design)
2. Find prompt category
3. Copy prompt section
4. Paste into generation tool (Midjourney, DALL-E, Runway, etc.)
5. Iterate and refine

### For GitHub Contributors
1. Update prompts in markdown files
2. Add new categories as needed
3. Document results (what worked, what didn't)
4. Create new .md files for specialized prompt categories

### For Integration
- Prompts are linked in `assets/` directories (see asset READMEs)
- Sync with visual asset outputs
- Cross-reference with content calendars
